# ⚡ JobSpark – Frontend Developer Assessment

This repository contains the job portal web application built for the **Brainic Tech Solutions Pvt Ltd Frontend Developer Assessment**.

## 📌 Project Links
- **Live App:** https://job-portal-web-application-wine.vercel.app/
- **GitHub Repo:** https://github.com/tanuja-badugu/Job-Portal-Web-Application

## 🧾 Assignment Summary
Built as a responsive job portal application using **HTML**, **CSS**, and **JavaScript**.

### Completed requirements
- Responsive navigation bar with modern UI
- Hero section with search bar
- Displayed 10 job cards (minimum requirement met)
- Search by job title and company name
- Filters by location and category
- Job details modal for each opening
- Professional footer
- Responsive design for mobile, tablet, and desktop

> The application is a static frontend project and does not require a build step.

## ✨ Features
- Clean hero layout with animated headline and search experience
- Live filtering and search of job data
- Glassmorphism-style panels and polished interaction states
- Responsive card grid for job listings and company highlights
- Theme toggle to switch between dark and light modes
- Accessible modal with keyboard close support

## 📁 Project Structure
```text
frontend/
```

## 🚀 Run Locally
Since this is a plain static frontend app, you can open it directly in the browser.

```bash
# Clone the repository
git clone https://github.com/tanuja-badugu/Job-Portal-Web-Application.git
cd Job-Portal-Web-Application

# Open index.html in your browser
# or use Live Server / a local static server
```
