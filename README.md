# ⚡ JobSpark – Responsive Job Portal

A fully animated, responsive Job Portal built with plain HTML, CSS, and JavaScript.

## 🚀 Live Demo
> Deploy to Netlify or Vercel and paste your link here.

## ✨ Features
- Animated hero section with floating shapes & counter animations
- Search jobs by title or company name (live filtering)
- Filter by location and category
- 10 job cards with smooth hover animations
- Job details modal with requirements, perks & apply button
- Sticky glassmorphism navbar with scroll effect
- Responsive for mobile, tablet, and desktop
- Dark theme with gradient accents

## 📁 Project Structure
```
frontend/
├── index.html   # Main HTML structure
├── style.css    # All styles & animations
├── script.js    # Job data, search, filter, modal logic
└── README.md
```

## 🛠 Setup
```bash
# Clone the repo
git clone https://github.com/<your-username>/jobspark.git
cd jobspark

# Open directly in browser
open index.html
# or use Live Server in VS Code
```

## 🌐 Deploy on Netlify
1. Push code to GitHub
2. Go to [netlify.com](https://netlify.com) → New site from Git
3. Select your repo → Deploy

## 🌐 Deploy on Vercel
```bash
npm i -g vercel
vercel
```

## 🧰 Tech Stack
- HTML5, CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (ES6+)
- No frameworks, no dependencies
